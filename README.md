
# 🤖 AI Practice Projects

This repository contains various AI and machine‑learning practice projects built using Python and popular AI libraries.  
Each subfolder contains a separate experiment or project focusing on a different AI concept, such as bots, data preprocessing, text‑to‑speech, automation, and more.

---

## 🧠 Projects Included

| Folder/File | Description |
|-------------|-------------|
| `Discord bot.py` | A Discord bot implementation (custom commands, automation) |
| `autosending.py` | Script for automated message sending workflows |
| `mani.py` | Miscellaneous practice Python script |
| `datasetpreprocessing.py` | Data preprocessing experiments |
| `english_telugu_tts_4gb.parquet` | Large dataset for English‑Telugu TTS |
| `chemistry/` | A project folder focused on chemistry data/logic |
| `info/` | Additional project info and support files |
| `config` | Configuration files |
| `hooks/` | Git hook scripts (if any) |
| `vercel.json` | Deployment configuration for Vercel |
| `PyWhatKit_DB.txt` | Text data used in experimentation |
| `Procfile` | Deployment process file (for Heroku, etc.) |
| Others | Various practice files and helper scripts |
